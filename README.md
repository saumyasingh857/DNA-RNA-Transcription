# DNA to RNA Transcription

This Python program converts a DNA sequence into its corresponding RNA sequence using complementary base pairing rules.

## Features
- Accepts DNA sequence as input.
- Converts DNA bases into RNA bases.
- Displays DNA and RNA sequences.

## Technologies Used
- Python

## Example

Input:
ATGC

Output:
DNA Sequence: ATGC
RNA Sequence: UACG
